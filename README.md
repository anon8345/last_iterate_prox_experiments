# Last iterate prox experiments
This repo contains results for experiments conducted using the BlockProx algorithm for distributed optimization with separable regularizers. Additional details regarding experimental setup can be found in
https://arxiv.org/pdf/2509.14488v1 section 6.1 and 6.2. We provide a brief summary of each figure.

## housing.png
Comparison of last and averaged BlockProx on sacramento housing dataset

## LS_1_1_20 and LS_2_1_20
Comparison of last and averaged BlockProx on synthesized network Lasso using l-1 regularizer (LS_1_1_20) and l-2 regularizer (LS_2_1_20) and one group with 20 nodes.

## LS_1_1_40 and LS_2_1_40
Comparison of last and averaged BlockProx on synthesized network Lasso using l-1 regularizer (LS_1_1_40) and l-2 regularizer (LS_2_1_40) and one group with 40 nodes which is fully connected.

## LS_1_5_75 and LS_2_5_75
Comparison of last and averaged BlockProx on synthesized network Lasso using l-1 regularizer (LS_1_5_75) and l-2 regularizer (LS_2_5_75) with five groups with 10, 17, 18, 18, 12 nodes respectively.
